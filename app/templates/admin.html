Overview

COSBELLE RAG Admin은 가상 코스메틱 제조사의 업무 데이터와 문서를 기반으로, Role과 보안 등급에 맞는 RAG 응답을 제공하는 엔터프라이즈 LLM Agent 운영 화면입니다. 관리자는 질의, 출처, 권한 체크, 로그와 대시보드를 한 흐름에서 확인할 수 있습니다.

Admin Foundations

RBAC 기반 RAG 운영을 위한 핵심 화면

Admin Chat

Role, Pre-check, Post-check 조건을 바꿔가며 RAG 응답과 차단 여부를 검증합니다.

Admin Sources

응답에 사용된 DB table, document citation, guard 결과를 관리자 관점에서 확인합니다.

Dashboards

Role별 접근 가능 데이터와 운영 지표를 대시보드 형태로 비교합니다.

SQL Logs

조회 시간, 질문, 실행 주체, 상태, SQL 상세를 기록하고 추적합니다.

Access Control

사용자 Role과 clearance를 기준으로 허용된 데이터만 조회되도록 검증합니다.

RAG Operations

Databricks Job 또는 API 백엔드와 연결해 검색, 답변 생성, 운영 로그를 관리합니다.
